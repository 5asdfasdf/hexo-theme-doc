# hexo-theme-doc-site

[User Documentation website](https://zalando-incubator.github.io/hexo-theme-doc) source for [hexo-theme-doc](https://github.com/zalando-incubator/hexo-theme-doc) package.

## Install

```bash
$ npm install hexo-cli -g && npm install
```

## Run

```bash
$ hexo s
```
